1. The project's name is Peddy
2. The project is basically a pet adoption website, where people can adopt pets like cats, dog, rabbits etc. Html, CSS, Javascript , Tailwind and Daisy UI has been used for this project
3. Key features-
   i. In the pets section, there are pet cards fetched from API, which shows image, name, breed, date of birth, gender, price of respective pets.
   ii. Modal pops up with pet details when details button is clicked.
   iii. Modal pops up with countdown od 3 seconds when adopt button is clicked.
   iv. Pets get sorted by price high to low when sort by price button is clicked.
   v. A loading spinner runs in pets section when data is being loaded.
4. ES6 Features-
   i. Arrow Functions
   ii. Fetching API
   iii. Template Literals(``)
   iv. Array methods ('.forEach()', '.sort()')
   v. Default Parameters and Call-back handlers
